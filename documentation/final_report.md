# Final Report – Museum Artwork Collection Analysis

This report summarizes the full workflow, insights, and outcomes of the project.

---

## 1. Introduction
The museum artwork dataset contains information about artists, artwork types, creation years, acquisition years, and demographics.  
The goal of this project was to clean the data, analyze it using SQL, and visualize insights using Power BI.

---

## 2. Methodology

### Step 1: Data Cleaning (SQL)
- Extracted numeric years  
- Standardized fields  
- Removed invalid entries  
- Created clean columns for analysis  

### Step 2: Data Analysis (SQL)
Answered five key questions:
1. How modern are the artworks?  
2. Which artists are featured the most?  
3. What types of artwork are most common?  
4. What are the acquisition trends?  
5. What does artist diversity look like?  

### Step 3: Visualization (Power BI)
- Built interactive dashboard  
- Added filters for exploration  
- Designed visuals for clarity  

---

## 3. Key Insights
- Modern artworks increased significantly after 1950  
- Ludwig Mies van der Rohe is the most featured artist  
- Photographs dominate the collection  
- Acquisition activity grew after the 1960s  
- Gender representation is uneven, with more male artists  

---

## 4. Conclusion
This project demonstrates strong skills in SQL, data cleaning, analytical thinking, and Power BI dashboard design.  
The final dashboard provides a clear, interactive view of the museum’s artwork collection.
