# Dashboard Description – Museum Artwork Analysis

This document describes the structure, visuals, and purpose of the Power BI dashboard created for the museum artwork dataset.

---

## Overview
The dashboard provides a visual summary of the museum’s artwork collection, answering five analytical questions using charts, cards, and filters.

---

## Dashboard Sections

### 1. Artwork Timeline
- Line chart showing artwork creation trends  
- Highlights growth in modern artworks  

### 2. Top Artists
- Bar chart ranking artists by artwork count  
- Shows most frequently featured creators  

### 3. Artwork Classification
- Donut or bar chart showing distribution of artwork types  
- Photographs, prints, paintings, sculptures, etc.  

### 4. Acquisition Trends
- Area or line chart showing acquisition activity over time  
- Identifies museum expansion periods  

### 5. Artist Diversity
- Stacked bar or pie charts for gender and nationality  
- Shows representation across the collection  

---

## Filters
- Artist  
- Classification  
- Year  
- Nationality  

These filters allow interactive exploration of the dataset.

---

## Purpose
The dashboard transforms SQL insights into a visual story, making it easier to understand patterns and trends in the museum’s collection.
