# Data Dictionary – Museum Artwork Dataset

| Column Name        | Description                                      |
|--------------------|--------------------------------------------------|
| artwork_id         | Unique identifier for each artwork               |
| title              | Name of the artwork                              |
| artist             | Name of the artist                               |
| nationality        | Artist nationality                               |
| gender             | Artist gender                                    |
| classification     | Type/category of artwork                         |
| date               | Year the artwork was created                     |
| acquisition_year   | Year the museum acquired the artwork             |
| clean_year         | Cleaned numeric year extracted from date column  |
