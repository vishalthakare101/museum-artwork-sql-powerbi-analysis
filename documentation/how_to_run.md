# How to Run This Project

## Requirements
- SQL Server / any SQL engine
- Power BI Desktop

## Steps
1. Run the SQL scripts in the /sql folder:
   - data_cleaning.sql
   - analysis_queries.sql

2. Load the cleaned dataset into Power BI.

3. Open the dashboard file:
   /powerbi/museum_artwork_dashboard.pbix

4. Explore the visuals to view insights for all five analytical questions.
