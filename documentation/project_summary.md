# Project Summary – Museum Artwork Collection Analysis

This project analyzes a large museum artwork dataset using SQL for data cleaning and insight generation, and Power BI for interactive visualization. The goal is to answer five analytical questions related to artwork trends, artist popularity, artwork types, acquisition patterns, and artist diversity.

The project demonstrates skills in:
- SQL data cleaning and transformation
- Writing analytical SQL queries
- Building a Power BI dashboard
- Creating visual insights from structured data
- Presenting results in a professional format

The final dashboard provides a clear, visual representation of all insights derived from SQL analysis.
